# ✈️ Travel Pick

![](https://i.imgur.com/avPOQGh.png)

> Travel Pick AI 여행지 추천 서비스 플랫폼<br>
> 임베딩 API를 활용해 유사도를 측정하여 여행지를 추천해주는 서비스
> Moderations API를 활용해 게시판 자동 관리가 가능한 서비스
> 
> 프로젝트 기간 : 2024.05.10 ~ 2024.05.23
